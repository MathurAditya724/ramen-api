### Resize & Optimize photos

Resize:

```plain
$ sips -Z 1200 yoshimuraya-001.jpg
```

Remove Exif information and optimize the image:

```plain
$ jpegtran -copy none -optimize -outfile yoshimuraya-001.jpg yoshimuraya-001.jpg
```
